# PROJ-39: Tooltips auf Icon-Buttons

## Goal
Add Tooltip labels to all icon-only buttons for accessibility and discoverability.

## User Stories
- As a user, I want tooltip hints on icon buttons so I know what they do

## Acceptance Criteria
- [ ] NotificationBell button wrapped in Tooltip
- [ ] ThemeToggle button wrapped in Tooltip
- [ ] Help link in sidebar has Tooltip
- [ ] All other icon-only buttons in navigation have Tooltips
- [ ] Uses shadcn/ui Tooltip component

## Technical Notes
Wrap icon-only buttons with TooltipProvider/Tooltip/TooltipTrigger/TooltipContent from @/components/ui/tooltip.
