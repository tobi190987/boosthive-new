# PROJ-45: Progress Indicators

## Goal
Step-based progress indicators for long operations like AI generation and keyword refresh.

## User Stories
- As a user, I want to see detailed progress during long AI or keyword operations so I know what's happening

## Acceptance Criteria
- [ ] AI generation shows step labels (Analyse, Generiere, Fertig)
- [ ] Keyword refresh shows step counter
- [ ] Steps auto-advance based on operation state
- [ ] Uses shadcn Progress component

## Technical Notes
Create ProgressSteps component; integrate with existing AI generator and keyword refresh flows.
